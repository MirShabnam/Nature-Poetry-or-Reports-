# Nature: Poetry or Report? — DH · Eprene

A clean, GitHub‑friendly build of the project (no spaces in filenames, Pages‑ready).  
This repo contrasts **poetic Nature** with **scientific Nature**, mapping motifs in literature to present‑day ecological indicators.

## Folders
- `docs/` → GitHub Pages site (open `docs/index.md`)
- `docs/assets/img/` → charts/images
- `data/` → CSV tables (facts & then‑now comparisons)
- `texts/` → TEI corpus (.xml)

## How to publish
1. Push to GitHub.
2. Settings → Pages → Source: `/docs` folder → Save.
3. Your site will appear at `https://<username>.github.io/<repo>/`.
