# Nature — From Poetry to Reports: A Digital Humanities Reflection

**Theme:** A Digital Humanities reflection on how humanity’s relationship with nature shifted from **poetic admiration** to **scientific documentation** — tracing the transformation from a world once **sung by poets** to one now **measured in reports of pollution and loss**.

_By Eprene_

## What this project does
- Builds a small comparative corpus of **poems** and **environmental reports**
- Runs **mixed‑method analysis**: word frequency, sentiment (lexicon‑based), lexical diversity, keyword extraction, co‑occurrence network
- Generates charts in `output/` and a Pages‑ready site in `docs/`

## How to run (locally)
```bash
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python code/analyze_nature_project.py
```
Outputs will appear in `output/` and embedded in `docs/index.md` (after you enable GitHub Pages).
